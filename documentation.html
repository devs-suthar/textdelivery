IT Technical documentation

Hardware and Software Configurations

ACME Corp uses four servers to run its software applications. The servers are configured as follows:

  • Server 1: Windows Server 2019, Intel Xeon processor, 32GB RAM, 500GB HDD
  • Server 2: Ubuntu Server 20.04, AMD Ryzen processor, 64GB RAM, 1TB SSD
  • Server 3: CentOS 8, Intel Core i7 processor, 16GB RAM, 2TB HDD
  • Server 4: Windows Server 2016, Intel Xeon processor, 128GB RAM, 1TB SSD

Workstations

ACME Corp uses four servers to run its software applications. The servers are configured as follows:

  • CPU: Intel Core i5 processor
  • RAM: 8GB
  • Storage: 256GB SSD
  • Operating System: Windows 10 Pro

Network Architecture

ACME Corp uses a local area network (LAN) to connect its devices. The LAN is connected to the internet via a fiber-optic connection. The LAN is divided into the following subnets:

  • Server 1: Windows Server 2019, Intel Xeon processor, 32GB RAM, 500GB HDD
  • Server 2: Ubuntu Server 20.04, AMD Ryzen processor, 64GB RAM, 1TB SSD
  • Server 3: CentOS 8, Intel Core i7 processor, 16GB RAM, 2TB HDD
  • Server 4: Windows Server 2016, Intel Xeon processor, 128GB RAM, 1TB SSD
